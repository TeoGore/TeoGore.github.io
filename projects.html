NLP WAF (bachelor thesis)

Description

Web Application Firewall (WAF) for web application protection, through request inspection using Natural Language Processing (NLP) techniques. The WAF is trained, using supervised learning, with data that are similar to a real world context: requests that "implements" some of the top 10 OWASP vulnerabilities in their payloads. There is also a simple vulnerable web app created for testing purposes.
